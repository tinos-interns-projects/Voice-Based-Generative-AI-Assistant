# Voice-Based Generative AI Assistant

## Overview
The Voice-Based Generative AI Assistant is an innovative project designed to facilitate voice interactions with generative AI models. It allows users to engage with AI in a more natural and intuitive manner, leveraging state-of-the-art speech recognition and natural language processing technologies.

## Features
- **Voice Activation**: Initiate interactions using voice commands.
- **Natural Language Processing**: Understand and respond to a wide range of queries in natural language.
- **Generative Responses**: Provide creative, coherent responses based on user input.
- **Multi-Language Support**: Engage with users in various languages.

## Installation
To install the Voice-Based Generative AI Assistant, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/muhammedmuflih/Voice-Based-Generative-AI-Assistant.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Voice-Based-Generative-AI-Assistant
   ```
3. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
To run the Voice-Based Generative AI Assistant, execute the following command:
```bash
python main.py
```

### Voice Commands
Here are some example voice commands you can use:
- "Hello Assistant"
- "What is the weather today?"
- "Tell me a joke"
